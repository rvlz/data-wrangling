## Installation
Clone projects
~~~
git clone git@github.com:rvlz/data-wrangling.git
~~~

Go to project folder
~~~
cd data-wrangling
~~~

Install virtual environment
~~~
python3 -m venv env
~~~

Activate virtual environment
~~~
. env/bin/activate
~~~

Install project dependencies
~~~
pip install -r requirements.txt
~~~

Launch
~~~
jupyter notebook
~~~
